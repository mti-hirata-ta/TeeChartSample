Supported browsers:
-------------------

TeeChart uses the basic Html5 <canvas> element available in
most modern browser versions or its equivalent polyfills.

-Microsoft Internet Explorer 9.0 and 10.0 (6.0 to 8.0: ExCanvas.js)
-Mozilla Firefox 3.6 and up
-Google Chrome 16.0 and up
-Firefox for Android
-Chrome for Android
-Apple Safari 5.0 and up
-Opera 11.6, Opera-Mini 5.0, Opera Mobile 10.0 and up
-Apple iOS Safari 3.2 and up
-Android WebKit 2.1 and up
-Dolphin Browser
-Maxthon Browser
